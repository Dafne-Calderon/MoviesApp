# MoviesApp

Aplicación móvil desarrollada en **Java** utilizando **Android Studio Otter 2025.2.1**, que permite explorar películas desde la API oficial de **The Movie Database (TMDB)**.  
La app muestra listas de películas populares, mejor valoradas y próximas a estrenarse, además de permitir realizar búsquedas por nombre y ver detalles de cada película.

---

## Tecnologías utilizadas

- **Java**
- **Android Studio Otter 2025.2.1**
- **Retrofit 2** + Gson  
- **Glide** (carga de imágenes)  
- **RecyclerView**  
- **ConstraintLayout**  
- **Material Design**
